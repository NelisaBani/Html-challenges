specific situations in which tables are routinely used.

Publishing:

A table of contents that is usually headed "contents" and it is a list found on a page before the start of a written work of its chapter or section titles or brief descriptions with their commencing page numbers.

Mathematics:

Mathematical tables are lists of numbers showing the results of calculation with varying arguments

Periodic Tables:

The periodic table of elements arranges all of the known chemical elements in an informative array. Elements are arranged from left to right and top to bottom in order of increasing atomic number. Order generally coincides with increasing atomic mass. The rows are called periods.

Database:

A database consists of one or more tables. Each table is made up of rows and columns. If you think of a table as a grid, the column go from left to right across the grid and each entry of data is listed down as a row. Each row in a relational is uniquely identified by a primary key.

Software Applications

Modern software applications give users the ability to generate, format, and edit tables and tabular data for a wide variety of uses
